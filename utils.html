def utils.mod_matrix (A B : matrixType) :
matrixType

Modular 2^32 addition of 4x4 matrices by doing Aᵢⱼ + Bᵢⱼ

The MOD operation (modulo 2^32 addition) is the key to make the salsa20 hash function irreversible. Everything is reversible except for this addition.
